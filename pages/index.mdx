# Home

We are building OpenLCH -- an open-source low cost humanoid robot designed for learning-based control.

Our mission is to *solve humanoid robotics for everyone while delivering shippable intermediaries*.

### Getting Started

Fork OpenLCH on GitHub. This is where all the development happens.
Join the community Discord. Along with Github, this is where developers cooridnate and dicuss development.
